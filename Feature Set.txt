Anatomy based features: 
- Protocol (http/s, ftp, ect.)
- Subdomain (www most common one)
- Top Level Domain
- Host Name: Subdomain + domain name + Top level Domain

NLP based features:
- Url containing '@' symbol (@ causes left side to be disregarded by browser) (yes/no)

Suspicious features:
- Is this domain simply an IP address (yes/no)
- Typosquated domain* (maybe)
- Presence of a legitimate url in the path (embedded domains) (yes/no)
- Is there presence of brand name within the url* (maybe)
